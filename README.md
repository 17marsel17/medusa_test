## What is Medusa

Medusa is a set of commerce modules and tools that allow you to build rich, reliable, and performant commerce applications without reinventing core commerce logic. The modules can be customized and used to build advanced ecommerce stores, marketplaces, or any product that needs foundational commerce primitives. All modules are open-source and freely available on npm.

## Currency Conversion API

This Medusa store includes a custom currency conversion endpoint that allows you to convert amounts between different currencies.

### Endpoint

```
GET /store/custom/currency/convert
```

### Parameters

- `amount` (required): The amount to convert (number)
- `from` (required): Source currency code (3-letter ISO 4217 code, e.g., USD)
- `to` (required): Target currency code (3-letter ISO 4217 code, e.g., EUR)

### Examples

**Convert 100 USD to EUR:**
```bash
curl "http://localhost:9000/store/currency/convert?amount=100&from=USD&to=EUR"
```

**Convert 50 EUR to GBP:**
```bash
curl "http://localhost:9000/store/currency/convert?amount=50&from=EUR&to=GBP"
```

### Response

```json
{
  "originalAmount": 100,
  "from": "USD",
  "to": "EUR",
  "convertedAmount": 85.42
}
```

### Features

- Real-time exchange rates from ExchangeRate-API
- Input validation for currency codes (ISO 4217 standard)
- In-memory caching for 1 hour to improve performance
- Error handling for invalid currencies and API failures
